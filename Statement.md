# Student Learning Analytics Dashboard

## Problem Statement

As a college student, I was struggling to keep track of my grades and study time across multiple subjects. I tried using notebooks, phone apps, and spreadsheets but nothing worked well for me.

### Problems I Had:
- My grades were scattered across different notebooks and apps
- I never knew how much time I was actually spending studying each subject
- Calculating grade averages by hand was tedious
- I kept losing my grade records
- I couldn't see which subjects needed more attention
- Everything was disorganized and I needed one central place for tracking

### What I Needed:
A simple tool to track grades, log study hours, and see my progress with visual charts. Something easy to use that wouldn't take forever to set up.

## Scope of the Project

### What This Project Does:
- Track grades for multiple subjects
- Log study session hours with dates
- Calculate grade averages automatically
- Show progress reports
- Create bar charts for grades and pie charts for study time
- Save data to files with backup system
- Delete study sessions if needed

### What This Project Doesn't Do:
- Web interface (desktop only)
- Multiple user accounts
- Advanced analytics or predictions
- Separate homework/test categories
- Calendar or scheduling features
- Cloud storage
- Mobile app version

### Technical Requirements:
- Uses 4 Python libraries (course requirement)
- Text file storage
- Command line interface
- Basic error handling

## Target Users

### Primary Users:
**College Students**
- Taking 4-6 subjects per semester
- Need simple grade tracking
- Want visual feedback on performance
- Basic computer skills
- Want to improve academic organization

### Secondary Users:
**High School Students**
- Managing multiple subjects
- Learning study habits
- Need progress monitoring

**Adult Learners**
- Part-time students
- Limited time for complex tools
- Taking certification courses

### User Needs:
- Quick and easy to use
- Visual progress tracking
- Help identifying weak subjects
- Organized record keeping
- Motivation through progress visualization

## High-Level Features

### 1. Student Profile
- Set student name
- Personalized dashboard
- Name appears on charts and reports

### 2. Grade Management
- Add subjects being studied
- Record grades (0-100 scale)
- Prevent duplicate subjects
- Input validation for grades

### 3. Study Time Tracking
- Log study hours per subject
- Automatic date recording
- View session history
- Delete incorrect entries
- Calculate total hours per subject

### 4. Progress Analytics
- Automatic grade average calculation
- Progress reports showing:
  - Average grade per subject
  - Total study hours per subject
  - Number of grades recorded
- Performance overview

### 5. Data Visualization
- Bar charts for grade averages
- Pie charts for study time distribution
- Color-coded charts
- Percentage labels
- Student name on chart titles

### 6. Data Management
- Automatic save on exit
- Manual save option
- Data loading on startup
- Backup file creation
- Text file format
- Data recovery system

### 7. User Interface
- Menu-driven navigation
- Emoji icons for visual appeal
- Clear feedback messages
- Error handling for invalid input
- Logical menu flow

### 8. Technical Features
- Cross-platform compatibility
- Minimal system requirements
- Single dependency (matplotlib)
- Reliable data storage
- Basic error recovery
